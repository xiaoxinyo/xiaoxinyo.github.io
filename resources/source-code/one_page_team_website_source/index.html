一页纸团队网站源码

2020-07-16 资源►源码 评论 字数统计: 526(字) 阅读时长: 2(分)

一款简约大气却不失细致的团队网站.

效果

  • 主页

    绿色字体轮播的内容在index.json文件中修改.
  • 服务

    标题,描述在index.json文件中修改.
  • 成员

    成员在member.json文件中修改.
  • 历程

    标题,内容在index.json文件中修改.
  • 关于

    工具列表,友情链接在public.json文件中修改.
  • 查询成员

    根据member.json文件查询.
  • 其他
    1. 网站标题,网站副标题,网站关键词,网站描述在core.json文件中修改.
    2. 网站图标修改./include/image/favicon.ico文件.
    3. 团队名称,团队描述,团队logo,团队QQ群号,团队QQ群链接在core.json文件中修改.
    4. json文件在./config/中.

配置

  • core.json
  • index.json
  • member.json
  • public.json
