PHP循环语句

2020-06-07 笔记►编程 评论 字数统计: 141(字) 阅读时长: 1(分)

条件

while

条件为真就会执行代码.

do…while

先执行后判断条件,如果为真则循环.

计数

for

根据写的次数确定循环次数.

foreach

适合用于数组循环,遍历数组中的每个键或值.
